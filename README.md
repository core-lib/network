Network
=======

Java HTTP / HTTPS Downloader Implementation

##> Installation

```
<repositories>
    ...
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
    ...
<repositories>

<dependencies>
    ...
    <dependency>
        <groupId>com.github.core-lib</groupId>
        <artifactId>network</artifactId>
        <version>v1.2.1</version>
    </dependency>
    ...
</dependencies>
```
